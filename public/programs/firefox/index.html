Introduction

Yes! You heard right this website has a public API. It can be used to receive personal data from the owner of this website as well as serverinformation from the server this webapplication is hosted. The documantion has some example code in different languages so its easy understandble for every developer. Enjoy!

Get started

The REST API provides public data accessible to any client aanonymously, as well as a authentification for the admin of the site only! For posting data to the server.

The REST API uses JSON exclusively as the request as well as response format, including errors. The responses are also fully dicoverable via hyperlinks when dealing with the responses.

Data Endpoint Reference

Resource Base Route Endpoint Info
About /api/about GET/POST Retrieve information about the website owner

const https = require('http');

https.get('https://floshodan.io/api/about', (resp) => {
  let data = '';

  resp.on('data', (chunk) => {
    data += chunk;
  });

  resp.on('end', () => {
    console.log(JSON.parse(data).explanation);
  });

  // Handle error
}).on("error", (err) => {
  console.log("Error: " + err.message);
});
import requests

response = requests.get('https://floshodan.io/api/about')
json_response = response.json()
print(json_response)
